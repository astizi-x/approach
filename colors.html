Палитра

Палитра IDM-PLUS обеспечивает узнаваемый и последовательный опыт во всех продуктах и материалах бренда.

Правила применения

01

Не заливайте всё голубым

Используйте Cyan только для ключевых акцентов и действий.

02

Контраст важнее декора

Текст и элементы интерфейса должны читаться на любом фоне.

03

Статусы не заменяют бренд-цвета

Семантические цвета используются для состояния, а не для бренда.

04

Один экран - один главный акцент

Выделяйте только один ключевой элемент с помощью Cyan.

Можно

  • Использовать Cyan только для ключевых действий.
  • Сохранять белое пространство вокруг акцентов.
  • Проверять контраст текста и интерфейсных элементов.
  • Применять серый для границ, меток и вторичного текста.

Нельзя

  • Заливать весь экран фирменным голубым.
  • Использовать акцентный цвет без смысловой роли.
  • Смешивать брендовые и статусные цвета.
  • Снижать читаемость ради декоративного эффекта.

Семантические цвета

Используются для передачи состояния. Не заменяют бренд-цвета.

03 / Palette

Палитра

Палитра ИДМ-ПЛЮС в approach поддерживает целостность пользовательского опыта во всех продуктах компании. Основу составляют три фирменных цвета: голубой #00BFF3, серый #9BA2A7 и белый #FFFFFF.

Нажмите на цвет, чтобы скопировать значение.

Роли цветов

РольЦветИспользование
Primary / Cyan#00BFF3Главный фирменный цвет ИДМ-ПЛЮС: действия, ссылки, активные состояния, акценты в презентациях и интерфейсах.
Neutral / Gray#9BA2A7Вторичный фирменный цвет: подписи, вспомогательный текст, технические метки, спокойные разделители и фоновые детали.
Surface / White#FFFFFFОсновная поверхность продуктов, документации и бренд-материалов. Белый фон сохраняет ощущение точности и чистоты.

Правила применения

01

Не заливать все голубым

Голубой должен вести пользователя к действию. Для больших поверхностей используем белый и мягкий холодный фон.

02

Контраст важнее декоративности

Текст должен читаться на любом устройстве. Не используем светло-синий текст на белом фоне для важных сообщений.

03

Статусы не конкурируют с брендом

Служебные цвета для ошибок, успеха и предупреждений допускаются только в интерфейсных состояниях и не входят в фирменную палитру ИДМ-ПЛЮС.

04

Один экран - один главный акцент

Если на экране несколько CTA, главным остается один. Остальные действия делаем вторичными.

Можно и нельзя

Можно

Использовать фирменные цвета по назначению

#00BFF3 — для акцентов и действий, #9BA2A7 — для вторичной информации, #FFFFFF — для чистой основной поверхности.

Нельзя

Подменять палитру служебными цветами

Не использовать цвета ошибок, успеха и предупреждений как фирменные. Они допустимы только для интерфейсных состояний.
